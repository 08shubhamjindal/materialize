Introduction

Chips can be used to represent small blocks of information. They are most commonly used either for contacts or for tags.

Jane Doe
Tag close

Contacts

To create a contact chip just add an img inside. 


  <div class="chip">
    <img src="images/yuna.jpg" alt="Contact Person">
    Jane Doe
  </div>

Tags

To create a tag chip just add an close icon inside. 


  <div class="chip">
    Tag
    <i class="material-icons">close</i>
  </div>

Basic Usage

Use the default chip template.

Use a custom chip template.

Use placeholders and override hint texts.

Markup


        <div class="chips-init"></div>

Initialize

Initialize chips


        $('.chips-init').material_chip({
          placeholder: 'Enter a tag',
          secondaryPlaceholder: '+Tag',
          data: [{
            tag: 'Apple',
          }, {
            tag: 'Microsoft',
          }, {
            tag: 'Google',
          }],
        });

Chip json


        var chip = {
          tag: 'chip content',
          image: '', //optional
          id: 1, //optional
        };

Events

Material chips exposes a few events for hooking into chips functionality.

Event Description
chips.add this method is triggered when a chip is added.
chips.delete this method is triggered when a chip is deleted.
chips.select this method is triggered when a chip is selected. 



        $('.chips-init').on('chip.add', function(e, chip){
          // you have the added chip here
        });

        $('.chips-init').on('chip.delete', function(e, chip){
          // you have the deleted chip here
        });

        $('.chips-init').on('chip.select', function(e, chip){
          // you have the selected chip here
        });
